# Copy Code

The `CopyCode` component allows users to copy code snippets to their clipboard with a simple click. It provides a user-friendly interface for copying code, enhancing the developer experience.

### Preview

<Preview className="mt-6">
  <CopyCode code="value" />
</Preview>

### Installation

```bash filename="Terminal" copy
npx shadcn@latest add "https://localhost:3000/r/button-copy.json"
```

### Usage

```tsx
import { CopyCode } from "@/components/alphaui/copy-code/copy-code";
```

```tsx
<CopyCode code="value" />
```

### Props

| Prop           | Type              | Default | Description                                                    |
| -------------- | ----------------- | ------- | -------------------------------------------------------------- |
| `code`         | `string`          | `-`     | The code to be copied to the clipboard.                        |
| `idleIcon?`    | `React.ReactNode` | `-`     | The icon to be displayed when the button is idle.              |
| `successIcon?` | `React.ReactNode` | `-`     | The icon to be displayed when the code is successfully copied. |
